A Flappy bird type game with a twist, meant to be played with two players
----------CONTROLS---------------
press W key for first bird
press L key for second bird
--------------------------------
Boss Fight at Score 20
revival mechanic which revives bird as long
as one bird is alive every 7 scores
------------------------------------
